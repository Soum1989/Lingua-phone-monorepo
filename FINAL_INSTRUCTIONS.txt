2. DEPLOY THE APPLICATION:
   Follow the manual deployment instructions in MANUAL_DEPLOYMENT_INSTRUCTIONS.md:
   
   a. Get cluster credentials:
      gcloud container clusters get-credentials lingua-cluster --zone=us-central1-a
   
   b. Build and push Docker images:
      docker build -t gcr.io/lingua-phone/lingua-backend:latest -f docker/backend.Dockerfile .
      docker push gcr.io/lingua-phone/lingua-backend:latest
      
      docker build -t gcr.io/lingua-phone/lingua-frontend:latest -f docker/frontend-k8s.Dockerfile .
      docker push gcr.io/lingua-phone/lingua-frontend:latest
   
   c. Apply Kubernetes manifests:
      kubectl apply -k k8s/
   
   d. Check pod status:
      kubectl get pods -n lingua-app

3. VERIFY SUCCESSFUL DEPLOYMENT:
   - Both frontend and backend pods should show "1/1 Running" status
   - Get the external IP: kubectl get services -n lingua-app
   - Access the application through the external IP

4. TROUBLESHOOTING:
   If the frontend pod is still crashing:
   - Check pod logs: kubectl logs <frontend-pod-name> -n lingua-app
   - Check pod description: kubectl describe pod <frontend-pod-name> -n lingua-app
   - Verify backend service: kubectl get services -n lingua-app
